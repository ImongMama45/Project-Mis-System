User Roles & Access Levels
This website supports three (3) user roles, each with distinct permissions:

**1. User**
Submit maintenance requests

Track the status of submitted requests

**2. Staff**
View and accept assigned requests

Manage and update their own requests

**3. Admin**
Full system access

Manage users, staff, and administrators

Oversee all requests and system data
